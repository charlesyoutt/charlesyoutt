# Charlie Youtt

I’m a Data Science & Finance student who enjoys building models and systems that reflect how decisions actually get made, especially when there are trade-offs and real constraints involved.

### Sailing Race Performance Prediction
End-to-end machine learning pipeline using real regatta data to predict sailing race outcomes with a Random Forest model.  
https://github.com/charlesyoutt/sailing-race-prediction

### Chicago Crime Analysis
Spatial and machine learning–based analysis of Chicago crime patterns using GeoPandas, KMeans clustering, and logistic regression.  
https://github.com/charlesyoutt/chicago-crime-spatial-analysis

### Workforce Optimization System
Python-based workforce optimization system using multi-objective scoring and evolutionary optimization to assign workers to tasks.  
https://github.com/charlesyoutt/workforce-optimization-system

### Fast Food Sankey Explorer
Interactive data visualization tool for exploring fast food nutrition patterns using Sankey diagrams and a web dashboard.  
https://github.com/charlesyoutt/fastfood-sankey-explorer
